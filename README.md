# Kuangan App
Aplikasi manajemen keuangan berbasis Flutter (masih dalam tahap pengembangan).

---

## Deskripsi
Kuangan adalah project aplikasi mobile yang sedang dikembangkan menggunakan Flutter.  
Saat ini, fitur yang sudah tersedia masih sebatas sistem autentikasi (login dan register).

---

## Progress Saat Ini
Fitur yang sudah diimplementasikan:
- Login user  
- Register user  
- Validasi input dasar  
- Penyimpanan data lokal sederhana
